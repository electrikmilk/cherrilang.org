### Set Background Sound

Set the background sound that plays.

```
enum backgroundSound {
    'BalancedNoise',
    'BrightNoise',
    'DarkNoise',
    'Ocean',
    'Rain',
    'Stream',
}

#define action 'setbackgroundsound' setBackgroundSound(backgroundSound ?sound: 'backgroundSound' = "BalancedNoise")
```
